# Diabetes Risk Prediction App

This project is a machine learning web application for predicting the risk of diabetes based on health indicators.

The application is built using:
- Streamlit (web interface)
- XGBoost (machine learning model)
- Joblib (model serialization)

## 📊 Project Description

The model predicts the probability of diabetes using user-provided health data such as:
- Age
- BMI (calculated from height and weight)
- Blood pressure
- Cholesterol level
- Physical activity
- Smoking status
- General health indicators

The prediction is based on a trained XGBoost model.
